Ex 2. FizzBuzz

Write a program that outputs all the numbers from 1 to 100. For numbers divisble by 3, print "Fizz" and for numbers divisible by 5, print "Buzz". For numbers divisible by both 3 and 5 output "FizzBuzz".


Ex 3. Chessboard

Write a program that creates a string that represents an 8x8 grid, using new-line characters to separate lines. At each position of the grid there is either a space ("X") or a "#" character. The characters should form a chess board. Output this string to the webpage.
